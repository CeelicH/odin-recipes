Lasagna Recipe


Description

This is a faster and less expensive lasagna that can be made using simple ingredients.

The goal here is to create a lasagna using ingredients totalling less than $10


Ingredients needed:

  • Mozarella Cheese
  • Pasta noodles
  • Anchovies
  • Green pepper
  • Sausage
  • Tomato sauce

Steps to make recipe

  1. Boil noodles
  2. Create pasta sauce
  3. Add in toppings
  4. Strain noodles and add back to pot
  5. Add sauce to noodles and serve
